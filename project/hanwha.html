한화생명금융서비스

금융 서비스 브랜드 웹사이트 구축 퍼블리싱

  • 기간 : 2020.10 ~ 2021.02
  • 역할 : 퍼블리싱
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 프로젝트 유형 금융 서비스 웹사이트 구축
  • 서비스 성격 기업 및 금융 정보 제공
  • 기여도 퍼블리싱 담당
  • 특이사항 신뢰성과 안정성 중심 UI

프로젝트 설명

한화생명금융서비스 웹사이트 구축 프로젝트로, 금융 서비스 특성에 맞게 신뢰감과 안정성을 최우선으로 고려한 UI 퍼블리싱을 담당했습니다. 다양한 금융 정보와 기업 콘텐츠를 명확하게 전달할 수 있도록 구조적인 마크업을 적용했습니다.

퍼블리싱 포인트

  • 금융 서비스에 적합한 정돈된 레이아웃과 정보 구조 설계
  • 콘텐츠 가독성을 높이기 위한 UI 요소 정렬 및 간격 조정
  • 브랜드 가이드를 준수한 안정적인 화면 구성
  • 반응형 웹을 적용해 다양한 디바이스 환경 대응
