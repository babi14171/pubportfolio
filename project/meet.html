LG Meet+

LG 화상회의 플랫폼 Meet+ 서비스 구축 퍼블리싱

  • 기간 : 2021.04 ~ 2023.03
  • 역할 : 퍼블리싱
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 프로젝트 유형 화상회의 플랫폼 구축
  • 대상 서비스 기업용 화상회의
  • 기여도 퍼블리싱 담당
  • 특이사항 기능 중심 UI 구성

프로젝트 설명

LG Meet+는 기업 환경에 특화된 화상회의 플랫폼으로, 다양한 기능을 직관적으로 전달할 수 있도록 UI 구조와 정보 위계를 고려한 퍼블리싱을 진행했습니다. 서비스 성격상 안정성과 가독성을 최우선으로 작업했습니다.

퍼블리싱 포인트

  • 복잡한 기능을 단계적으로 이해할 수 있도록 정보 구조 설계
  • 서비스 소개 및 기능 안내 화면 중심의 UI 퍼블리싱
  • 다양한 해상도에서도 일관된 사용성을 유지하는 반응형 대응
  • 브랜드 가이드를 준수한 안정적인 UI 구현
