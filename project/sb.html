SOFTBRIDGE 시스템 구축

기업용 시스템 구축 프로젝트 퍼블리싱

  • 기간 : 2020.08 ~ 2020.09
  • 역할 : 퍼블리싱
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 소프트브리지 바로가기
  • 프로젝트 유형 기업 내부 시스템 구축
  • 서비스 성격 업무 지원 시스템
  • 기여도 퍼블리싱 담당
  • 특이사항 SI 프로젝트 특성 반영

프로젝트 설명

소프트브리지에서 진행한 기업용 시스템 구축 프로젝트로, 실제 업무에 활용되는 화면 중심의 퍼블리싱을 담당했습니다. 기능 위주의 화면 구성과 데이터 가독성을 고려하여 안정적인 UI 구현에 집중했습니다.

퍼블리싱 포인트

  • 데이터 중심 화면에 적합한 테이블·리스트 UI 구성
  • 업무 흐름을 방해하지 않는 안정적인 레이아웃 설계
  • 반복 사용되는 UI 요소의 구조 정리 및 일관성 유지
  • 다양한 해상도 및 브라우저 환경 대응
