이연제약 사이트 구축

제약 기업의 신뢰성과 전문성을 강조한 공식 웹사이트 구축 프로젝트

  • 기간 : 2020.03 ~ 2020.06
  • 역할 : 퍼블리싱 전담
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 이연제약 바로가기
  • 프로젝트 유형 제약회사 기업 홈페이지 구축
  • 주요 목적 기업 이미지 개선 및 정보 전달
  • 기여도 퍼블리싱 100%
  • 특이사항 의료·제약 콘텐츠 가이드라인 준수

프로젝트 설명

이연제약의 기업 신뢰도를 강화하기 위한 공식 웹사이트 구축 프로젝트로, 제약 산업 특성상 정확한 정보 전달과 안정적인 UI 구성을 중심으로 퍼블리싱을 진행했습니다. 연구·개발, 제품 정보, 기업 소개 등 다양한 콘텐츠를 사용자가 쉽게 이해할 수 있도록 구조화했습니다.

퍼블리싱 포인트

  • 제약·의료 분야 특성을 고려한 차분하고 신뢰감 있는 UI 구현
  • 다량의 텍스트 콘텐츠를 고려한 가독성 중심 레이아웃 설계
  • 연구·제품 정보 페이지의 정보 구조 정리
  • 반응형 웹 대응을 통한 다양한 디바이스 접근성 확보
  • 운영 및 유지보수를 고려한 마크업 구조 정리
