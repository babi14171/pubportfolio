  • 기술 : HTML / CSS / JavaScript / jQuery

프로젝트 설명

풀무원 브랜드 공식 웹사이트의 장기 운영 프로젝트로, 정기적인 콘텐츠 업데이트와 이벤트 페이지 제작, UI 수정 및 유지보수를 담당했습니다. 브랜드 가이드라인을 준수하면서도 사용자 경험을 해치지 않는 방향으로 퍼블리싱을 진행했습니다.

퍼블리싱 포인트

  • 브랜드 가이드에 맞춘 UI 유지 및 일관성 관리
  • 이벤트·프로모션 페이지의 빠른 퍼블리싱 대응
  • 운영 중 발생하는 UI 이슈 및 크로스 브라우징 대응
  • 기존 마크업 구조를 이해하고 안정적으로 유지보수
  • 디자인·기획·개발팀과의 협업을 통한 수정 반영
