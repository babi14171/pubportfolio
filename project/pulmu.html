한마음평화연구재단의 공식 웹사이트 구축 프로젝트로, 재단의 목적과 활동 내용을 명확하게 전달하는 데 중점을 두고 퍼블리싱을 진행했습니다. 공익 사이트 특성상 정보의 가독성과 안정적인 화면 구성을 가장 중요하게 고려했습니다.

퍼블리싱 포인트

  • 정보 전달 중심의 단순하고 명확한 레이아웃 구성
  • 연구·활동 콘텐츠를 체계적으로 보여주는 콘텐츠 구조 설계
  • 다양한 연령층을 고려한 가독성 중심 UI 구성
  • 반응형 웹을 적용해 디바이스 환경에 유연하게 대응
