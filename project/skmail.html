SK 메일 시스템

기업 내부 메일 시스템 구축 퍼블리싱

  • 기간 : 2020.08 ~ 2020.09
  • 역할 : 퍼블리싱
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 프로젝트 유형 기업 내부 시스템 구축
  • 서비스 성격 메일 및 커뮤니케이션 시스템
  • 기여도 퍼블리싱 담당
  • 특이사항 보안 및 안정성 중심 UI

프로젝트 설명

SK 내부에서 사용하는 메일 시스템 구축 프로젝트로, 업무 효율을 높이기 위한 UI 퍼블리싱을 담당했습니다. 실제 실무자가 장시간 사용하는 시스템 특성을 고려해 가독성과 안정성을 최우선으로 구현했습니다.

퍼블리싱 포인트

  • 메일 목록·읽기·작성 화면 등 핵심 기능 중심 UI 구성
  • 데이터 중심 화면에서도 피로도가 낮은 레이아웃 설계
  • 업무 흐름을 방해하지 않는 안정적인 인터페이스 구현
  • 다양한 브라우저 환경에서도 일관된 UI 유지
