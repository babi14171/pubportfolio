  • 기간 : 2020.10 ~ 2020.12
  • 역할 : 퍼블리싱
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 눈높이 바로가기
  • 프로젝트 유형 교육 서비스 웹사이트 구축
  • 서비스 성격 학습 콘텐츠 및 브랜드 소개

  • 특이사항 학부모·학생 대상 UI 고려

프로젝트 설명

대교 눈높이의 공식 웹사이트 구축 프로젝트로, 학부모와 학생이 주요 사용자라는 점을 고려해 정보 전달이 명확하고 신뢰감 있는 UI 구현에 집중했습니다. 브랜드 이미지를 유지하면서도 학습 콘텐츠를 쉽게 이해할 수 있도록 퍼블리싱을 진행했습니다.

퍼블리싱 포인트

  • 교육 서비스 특성에 맞춘 안정적이고 친근한 UI 구성
  • 학습 콘텐츠 및 프로그램 정보를 체계적으로 정리한 레이아웃
  • 연령대가 다른 사용자도 쉽게 이용할 수 있는 가독성 확보
  • PC·모바일 환경을 고려한 반응형 웹 퍼블리싱
