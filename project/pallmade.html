폴메이드 (Pallmade)

  • 기간 : 2020.09 ~ 2021.12
  • 역할 : 퍼블리싱
  • 기술 : HTML / CSS / JavaScript / jQuery
  • 폴메이드 바로가기
  • 프로젝트 유형 브랜드 웹사이트 구축
  • 서비스 성격 브랜드 소개 및 콘텐츠 중심

  • 특이사항 브랜드 이미지 강조

프로젝트 설명

폴메이드 공식 웹사이트 구축 프로젝트로, 브랜드의 정체성과 이미지를 효과적으로 전달할 수 있도록 비주얼 중심의 UI 퍼블리싱을 담당했습니다. 콘텐츠 구성과 레이아웃 흐름을 고려하여 브랜드 스토리가 자연스럽게 전달되도록 구현했습니다.

퍼블리싱 포인트

  • 브랜드 이미지를 강조한 비주얼 중심 레이아웃 구성
  • 콘텐츠 흐름에 맞춘 섹션 구성 및 인터랙션 구현
  • PC·모바일 환경 모두에서 브랜드 경험을 유지하는 반응형 대응
  • 디자인 의도를 정확히 구현하는 마크업 구조 설계
